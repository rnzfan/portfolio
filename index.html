    Customer Communication Hub

    This is a light weight customer communication hub web application. It was designed for a team to use to build customer relationships with every conversation in a human way. The application was implemented to synchronize with a real Gmail Account which acts like an actual company primary contact email address. All team members are the Users of the application. The application is the frontpage for all team members to manage the conversations with the customers. Each incoming email will be transformed as a Task in the application.
